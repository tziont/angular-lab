State Management without NgRx

In this section, we’ll explore Angular-native approaches to managing application state without relying on heavy libraries like NgRx. You’ll learn how to use services, RxJS, and the new Signals API to create clean, scalable, and testable state layers.

We'll cover managing local state using Signals inside components, lifting shared state into services using RxJS or Signals, and applying global state patterns — all while handling loading, success, and error flows.

You’ll also compareSignals and BehaviorSubjects to understand when and why to use each.
