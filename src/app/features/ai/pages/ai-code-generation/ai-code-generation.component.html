Using AI Tools in VS Code

Follow these steps to generate or refine this component with AI:

  1. GitHub Copilot: Open user-list.component.ts, start typing a method like ngOnInit() or getUsers(), and let Copilot suggest the implementation. Accept suggestions or modify as needed.
  2. Copilot Chat: Open the chat sidebar, select your component code, and ask for improvements or explanations. Example prompt: "Refactor this UserListComponent to use proper Angular best practices."
  3. Codeium: Type the first few characters of your function or service in the editor. Accept inline suggestions to complete it faster.
  4. ChatGPT VS Code Plugin: Highlight a component or service, then ask the AI to generate an alternative implementation or add comments. Compare with your current code.

Compare AI Output vs Manual Improvement

  • Check for naming consistency, folder structure, and type correctness.
  • Ensure Angular best practices are followed (e.g., OnInit usage, proper subscriptions).
  • Manually refine formatting, comments, and error handling.
  • Save both versions to see what AI gets right automatically and where manual improvements are necessary.
