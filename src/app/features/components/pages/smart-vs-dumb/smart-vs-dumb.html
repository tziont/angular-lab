Smart vs Dumb Components

Smart (Container)

Handles application logic, manages state, and communicates with services.

  • Fetches and prepares data
  • Manages state
  • Handles side effects

Dumb (Presentational)

Focuses purely on UI. Receives data via inputs and emits events.

  • Uses @Input() for data
  • Uses @Output() for events
  • No business logic
