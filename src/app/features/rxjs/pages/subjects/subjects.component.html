Subject and Its Variants

A Subject in RxJS is a special kind of Observable that you can also send values into. It's useful when you want to manually control the data being shared across multiple subscribers. Unlike regular observables that only emit data from within, a Subject lets you both listen and emit.

RxJS provides a few types of Subjects, each with slightly different behavior:

  • Subject: Only sends values to subscribers who are already subscribed when the value is emitted.
  • BehaviorSubject: Always remembers the most recent value and sends it immediately to new subscribers.
  • ReplaySubject: Remembers a set number of previous values and replays them to any new subscribers.
  • AsyncSubject: Waits until it's completed, then sends only the last value to all subscribers.
