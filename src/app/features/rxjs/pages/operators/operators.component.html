Common RxJS Operators

RxJS Operators are powerful tools that help you transform, filter, combine, and manage streams of data. They make it easier to work with asynchronous data in a declarative way.

Below are some of the most frequently used operators in Angular development:

  • map: Transforms each emitted value using a projection function.
  • filter: Emits only values that satisfy a given condition.
  • tap: Performs side effects like logging or debugging, without changing the stream.
  • switchMap: Switches to a new inner observable, canceling the previous one. Useful for typeahead inputs or auth checks.
  • concatMap: Maps each value to an inner observable and waits for it to complete before moving to the next. Great for sequential HTTP requests.
  • mergeMap: Like concatMap, but processes all inner observables concurrently. Ideal for parallel requests.
  • catchError: Catches and handles errors in the observable chain.
